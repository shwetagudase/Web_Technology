DOM: document Object model
DOM is the bridge between HTML and js
It is Programming interface.
When Brower loads an html page It converts HTML into an Object that Object Struct is called DOM.
JS uses DOM to read,change and delete HTML elements.

DOM Tags
activity 1: write the uses of following:
document.getElementById("title) // to access the element we can change text, style, class, etc.
document.getElementById("title).className //Gets or sets the class name of the selected element.
document.getElementById("title).setAttributes//Adds or updates an attribute of an element.
document.getElementById("title).getAttributes//Returns the value of a specified attribute.
document.getElementById("title).innerHTML//get or set the HTML content inside an element
